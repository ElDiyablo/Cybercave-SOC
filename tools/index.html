SOC Analyst Toolkit

Tools used in CYS 650 and summer SOC operations — organized by analyst workflow phase.

// 01 — SIEM & Log Analysis

Core Monitoring Platform

ToolCategoryWhat We Use It ForLink
Kibana SIEM Primary analyst dashboard — search logs, visualize threat trends, triage alerts from PA firewall events. elastic.co
Elasticsearch SIEM Backend index that powers Kibana searches. Understanding index patterns and field mappings helps with advanced queries. elastic.co

Intel & Enrichment

// 03 — Detection & Frameworks

Detection Engineering & References

🎯

MITRE ATT&CK

The definitive knowledge base of adversary tactics, techniques, and procedures. Map log observations to known attacker behavior and attack stages.

→ attack.mitre.org

🔍

Awesome Threat Detection

Curated GitHub list of tools, papers, and resources for threat detection and hunting — rule repositories, analytics frameworks, and playbooks.

→ GitHub: 0x4D31/awesome-threat-detection

📐

Sigma Rules

Generic detection rule format convertible to KQL, Splunk SPL, and other SIEM languages. Pre-built detection logic for common attack patterns.

→ GitHub: SigmaHQ/sigma

🗺

MITRE D3FEND

Defensive counterpart to ATT&CK — a knowledge graph of cybersecurity countermeasures mapped to offensive techniques.

→ d3fend.mitre.org

🧩

MITRE CAR

Cyber Analytics Repository — detection analytics mapped directly to ATT&CK techniques with pseudocode adaptable to KQL queries.

→ car.mitre.org

📋

NIST Cybersecurity Framework

The five functions — Identify, Protect, Detect, Respond, Recover — provide the structural vocabulary for SOC operations.

→ nist.gov/cyberframework

// 04 — Network & Investigation

Network Analysis & Investigation

ToolCategoryUse CaseLink
Wireshark Network Packet-level analysis. Inspect PCAP evidence, decode protocols, follow TCP streams during deeper investigations. wireshark.org
NetworkMiner Network Passive network forensics — extracts hosts, credentials, and files from PCAP without deep protocol knowledge. netresec.com
CyberChef Analysis Browser-based data transformation — decode base64, analyze obfuscated payloads, parse hex, extract IOCs. gchq.github.io
WHOIS / DomainTools Recon Domain registration history, registrant data, and hosting provider info for suspicious domains in PA logs. domaintools.com
